PAT甲级1002 A+B Polynomials

This time, you are supposed to find A+B where A and B are two polynomials.

Input Specification:

Each input file contains one test case. Each case occupies 2 lines, and each line contains the information of a polynomial（多项式）:
K N1 a​N1 N​2 aN2… NK aNK
where K is the number of nonzero terms in the polynomial, N​i​ and aNi​​ (i=1,2,⋯,K) are the exponents and coefficients, respectively. It is given that 1≤K≤10，0≤NK<⋯<N2<N1≤1000.

输入K为这个多项式有几项，接下来的每两个数为一项，第一个数为指数，第二个数为指数

Output Specification:

For each test case you should output the sum of A and B in one line, with the same format as the input. Notice that there must be NO extra space at the end of each line. Please be accurate to 1 decimal place.

输出两个多项式相加以后有几项，并输出指数和系数，注意数据要精确到小数点后一位

Sample Input:
2 1 2.4 0 3.2

2 2 1.5 1 0.5
Sample Output:
3 2 1.5 1 2.9 0 3.2

代码

#include<iostream>
#include<map>
#include<iterator>
using namespace std;

std::map<int,double,greater<int> > poly;
	
int main(){
	int count=2,K,n;
	double m=0;
	while(count>0){
		cin >> K;
		for(int i=0;i<K;i++){
			cin >> n >> m;
			if(poly.count(n) == 0){
				poly.insert(std::pair<int,double>(n,m));
			}else{
				m = poly.at(n) + m;
				poly.erase(n);
				poly.insert(std::pair<int,double>(n,m));
			}
			if(poly.at(n)==0){
			//如果系数为0，那么删除此数据项
			//一开始我没有注意题目里面的nonzero terms没有加这一段导致没有AC
				poly.erase(n);
			}
		}
		count--;
	}
	cout << poly.size();
	map<int,double>::iterator it = poly.begin();
	while(it != poly.end()) {
		printf(" %d %.1lf",it->first,it->second);
		it ++ ;
	}
}

要点

  1. map从大到小排序
std::map<int,double,greater<int> > map
  1. map插入
map.insert(std::pair<int,double>(key,value));
