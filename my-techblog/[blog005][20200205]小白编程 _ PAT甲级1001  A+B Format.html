PAT甲级1001 A+B Format

Calculate a+b and output the sum in standard format – that is, the digits must be separated into groups of three by commas (unless there are less than four digits).

Input Specification:

Each input file contains one test case. Each case contains a pair of integers a and b where −106 ≤a,b≤ 106. The numbers are separated by a space.

输入两个int数字（−106 ≤a,b≤ 106），中间用空格分隔。

Output Specification:

For each test case, you should output the sum of a and b in one line. The sum must be written in the standard format.

使用指定格式输出两个数之和：每三位用“,”分隔。

Sample Input:
-1000000 9
Sample Output:
-999,991

代码（两个写法）

#include<iostream>
#include<string>
#include <cstring>
/* 1001 A+B Format */

using namespace std;

//方法一
//*/
int main(){
	int a,b,c,i,j;
	scanf("%d %d",&a,&b);
	c = a + b;
	char s1[100],s2[100]; 	//s1用作承接c,s2用作去除'-' 
	sprintf(s1,"%d",c);
	if(s1[0]=='-'){
		printf("-");	//负号在此先输出 
		for(i=0;i<strlen(s1)-1;i++){
			s2[i] = s1[i+1];		//把正数的s1赋值给s2
		}
	}else{
		strcpy(s2,s1);		//正数情况下把s1完全赋值给s2 
	}
	int len = strlen(s2);	//计算数字字符串长度 
	for(j=0;j<len;j++){
		if((len-j) % 3 == 0){	//当其剩下3的倍数位是输出逗号 
			if(len - j != len){ 	//为保证6位数时第一位输出不是逗号 
				printf(",");
			}
		}
		putchar(s2[j]);		//输出数字 
	}
	return 0;
}
//*/

要点

  1. strlen()包含在头文件#include<string>中
  2. 在逻辑判断是否是3的倍数来输出逗号时，要保证数字长度为3的倍数时第一位不输出逗号
